# Burgers

# Description

A simple app where user can input names of any burger they want. It then generates into the "To Be Eaten" column. User is allowed to click "Devour" to shift the burger to the "Eaten" column.

# Tools/Methods Used

MySQL2
Heroku
Nodejs
Express
Express-Handlebars
